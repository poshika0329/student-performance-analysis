# Student Performance Analysis

## Objective
To analyze student performance data and identify patterns and relationships between different subjects.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Steps Performed
- Data creation and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Insight generation

## Key Insights
- Strong correlation between reading and writing scores
- Math scores show higher variation
- Gender-based differences observed in performance

## Conclusion
This project demonstrates how data analysis can be used to extract meaningful insights from student performance data.
